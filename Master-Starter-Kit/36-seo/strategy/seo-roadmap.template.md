## Maturity Tier Assessment

Before planning, determine your current tier. Be honest — starting at a tier above your actual maturity creates a broken foundation.

### Tier Assessment Checklist

**Check all that apply:**

#### Technical Foundation
- [ ] Google Search Console is set up and verified
- [ ] Google Analytics 4 is tracking organic traffic
- [ ] XML sitemap exists and is submitted
- [ ] Robots.txt is correctly configured
- [ ] HTTPS is enabled site-wide
- [ ] Core Web Vitals pass on mobile and desktop
- [ ] No critical crawl errors (< 5)
- [ ] Structured data implemented on key pages

#### Content Foundation
- [ ] 10+ keyword-targeted pages published
- [ ] At least 1 complete topic cluster (pillar + 5 cluster posts)
- [ ] Content-keyword map exists and is maintained
- [ ] Content is being published at least 2x/month consistently
- [ ] No significant keyword cannibalization issues

#### Authority
- [ ] Domain Rating > 15 (Ahrefs)
- [ ] 10+ referring domains from unique sites
- [ ] Active link building program in place
- [ ] Some pages ranking on page 1 for target keywords

#### Process
- [ ] Keyword research is completed and documented
- [ ] SEO reporting happens monthly
- [ ] Competitive intelligence is tracked quarterly
- [ ] Content refresh schedule exists

### Scoring

| Checks Passed | Tier |
|---------------|------|
| 0-4 | **Tier 1: Foundation** — You are building from scratch |
| 5-9 | **Tier 2: Growth** — Foundation exists, time to grow |
| 10-14 | **Tier 3: Scale** — Growth engine is running, time to accelerate |
| 15+ | **Tier 4: Defend and Expand** — Mature program, protect and expand |

## Tier 1: Foundation

**Duration**: Months 0-3
**Goal**: Build the technical, content, and measurement foundations. No organic traffic growth expected.
**Mindset**: "Make the site worthy of ranking before trying to rank."

### Quarter Plan: Tier 1

#### Month 1: Technical Foundation and Research

| Week | Activity | Deliverable | Owner |
|------|----------|-------------|-------|
| 1 | Set up tracking and tools | GSC verified, GA4 configured, rank tracking tool active | Dev / SEO |
| 1 | Technical SEO audit | Audit report with prioritized issue list | SEO |
| 2 | Fix critical technical issues | XML sitemap, robots.txt, HTTPS, canonical tags | Dev |
| 2 | Keyword research — seed generation | 30+ seed keywords documented | SEO |
| 3 | Keyword research — expansion and prioritization | 100+ keywords scored and prioritized | SEO |
| 3 | Competitor identification | Top 5 SEO competitors identified | SEO |
| 4 | Content-keyword map | Map of existing pages to primary/secondary keywords | SEO |
| 4 | Topic cluster plan | 2-3 clusters identified with pillar + cluster post topics | SEO |

#### Month 2: On-Page Optimization and First Content

| Week | Activity | Deliverable | Owner |
|------|----------|-------------|-------|
| 1-2 | Optimize existing pages | Title tags, meta descriptions, headings, internal links for all existing pages | SEO / Content |
| 2 | Fix remaining technical issues | Core Web Vitals passing, no crawl errors | Dev |
| 3 | Publish pillar page #1 | Comprehensive pillar page (3,000-5,000 words) | Content |
| 3 | Implement structured data | Schema markup on key pages (Organization, Product, FAQ) | Dev |
| 4 | Publish 2 cluster posts | Supporting content for pillar #1 | Content |
| 4 | Internal linking overhaul | All published pages linked according to cluster architecture | SEO |

#### Month 3: Content Velocity and First Outreach

| Week | Activity | Deliverable | Owner |
|------|----------|-------------|-------|
| 1 | Publish 2 more cluster posts | 4 total cluster posts supporting pillar #1 | Content |
| 2 | Begin pillar page #2 | Second comprehensive pillar page | Content |
| 2 | Start link building outreach | First 10 outreach emails sent (resource pages, guest posts) | SEO |
| 3 | Publish 2 more pages | Cluster posts or standalone keyword-targeted pages | Content |
| 4 | Month-3 SEO report | Traffic, rankings, indexed pages, issues found | SEO |
| 4 | Roadmap review | Assess progress, adjust Tier 2 plan | SEO |

## Priority Matrix

When capacity is limited, use this matrix to decide what to do first.

### Priority Levels

| Level | Criteria | Examples |
|-------|----------|---------|
| **P0 — Critical** | Blocks all other SEO work. Revenue impact if not fixed. | Site down, pages deindexed, major crawl errors, manual penalty |
| **P1 — High** | Directly impacts rankings or conversions. | Broken internal links on top pages, missing title tags, CWV failures, keyword cannibalization |
| **P2 — Medium** | Improves SEO performance but not blocking. | New content publication, link building outreach, schema implementation, content refreshes |
| **P3 — Low** | Nice to have, marginal impact. | Minor formatting fixes, non-critical schema types, social meta tags, breadcrumb optimization |

### Priority Decision Tree

```
Is it preventing pages from being indexed or crawled?
├── Yes → P0 CRITICAL — fix immediately
└── No  → Is it directly hurting an existing ranking?
          ├── Yes → P1 HIGH — fix this week
          └── No  → Does it directly create new ranking potential?
                    ├── Yes → P2 MEDIUM — schedule in current sprint
                    └── No  → P3 LOW — add to backlog
```

### Priority Matrix by Activity Type

| Activity | Tier 1 Priority | Tier 2 Priority | Tier 3 Priority | Tier 4 Priority |
|----------|----------------|----------------|----------------|----------------|
| Fix crawl errors | P0 | P0 | P0 | P0 |
| Fix Core Web Vitals | P1 | P1 | P1 | P1 |
| Keyword research | P1 | P2 | P2 | P3 |
| Publish new content | P1 | P1 | P1 | P2 |
| Optimize existing pages | P1 | P2 | P2 | P1 |
| Build backlinks | P3 | P1 | P1 | P2 |
| Content refresh | N/A | P3 | P2 | P1 |
| Competitive analysis | P2 | P2 | P2 | P1 |
| Schema markup | P2 | P2 | P3 | P3 |
| International SEO | N/A | N/A | P3 | P2 |
| Content pruning | N/A | N/A | P3 | P2 |

## Dependency Map

Some SEO activities depend on others. Doing them out of order wastes effort.

### Critical Dependencies

```
    │ 3a. On-Page      │ │ 3b. Content   │ │ 3c. Topic Cluster│

                    └──────────┬───────────┘
                               │
                    ┌──────────▼───────────┐
                    │ 6. Content Refresh   │
                    │    & Expansion       │
                    │    (need published   │
                    │     content + data)  │
                    └──────────────────────┘
```

### Dependency Rules

| Activity | Depends On | Must Complete Before |
|----------|-----------|---------------------|
| Keyword research | GSC and analytics setup | Content creation, on-page optimization |
| Content creation | Keyword research, topic cluster plan | Link building |
| On-page optimization | Keyword research | Link building (optimize before promoting) |
| Link building | Content exists to link to | N/A (ongoing) |
| Content refresh | 3+ months of ranking data | N/A (ongoing) |
| Technical SEO | Nothing (can start immediately) | Everything else |
| Schema markup | Content published on pages | N/A |
| Competitive analysis | Nothing (can start immediately) | Keyword research, content planning |

### Common Sequencing Mistakes

| Mistake | Why It Fails | Correct Order |
|---------|-------------|---------------|
| Link building before content exists | Nothing to link to, wasted outreach | Publish content first, then build links |
| Content before technical fixes | Pages may not be indexed or crawled | Fix technical issues first |
| Scaling content before keyword research | Content targets wrong keywords | Research first, then create |
| Content refresh before 6 months of data | Not enough data to know what to improve | Wait for data, then refresh |
| Programmatic SEO before manual content | Need manual content to prove templates work | Prove with manual content first |

## Staffing Recommendations

### By Tier

| Tier | Minimum Team | Ideal Team | Notes |
|------|-------------|-----------|-------|
| **Tier 1** | Solo founder or 1 SEO-aware person | 1 SEO lead (part-time OK) + dev support | At this stage, learning SEO fundamentals is as important as execution |
| **Tier 2** | 1 SEO lead + 1 content writer | 1 SEO lead + 1-2 content writers + part-time dev | Consistent content production is the bottleneck |
| **Tier 3** | 1 SEO lead + 2 writers + dev support | SEO lead + content manager + 2-3 writers + dev + link builder | Specialization starts to matter |
| **Tier 4** | SEO lead + content manager + 2 writers + link builder | SEO director + content manager + 3-4 writers + editor + dev team + PR/outreach + analyst | Full team with clear specializations |

### Role Definitions

| Role | Responsibilities | When to Hire |
|------|-----------------|-------------|
| **SEO Lead** | Strategy, keyword research, technical audits, reporting | Tier 1 (can be founder) |
| **Content Writer** | Drafting keyword-targeted content | Tier 2 |
| **Content Editor** | Quality control, consistency, SEO review of content | Tier 3 |
| **Content Manager** | Editorial calendar, writer management, content strategy | Tier 3 |
| **Link Building Specialist** | Outreach, guest posting, digital PR, relationship building | Tier 3 |
| **Technical SEO Dev** | Schema implementation, site speed, crawl optimization | Tier 3 (part-time from Tier 1) |
| **SEO Analyst** | Reporting, attribution modeling, competitive intelligence | Tier 4 |

### In-House vs. Agency vs. Freelance

| Approach | Best For | Tier Fit | Typical Cost |
|----------|---------|----------|-------------|
| **DIY (founder)** | Learning, bootstrapped startups | Tier 1 | Time only + $100-200/mo tools |
| **Freelance SEO consultant** | Strategy + periodic audits | Tier 1-2 | $1,000-5,000/mo |
| **Freelance writers** | Content production scaling | Tier 2-3 | $200-800 per article |
| **SEO agency** | Full-service execution | Tier 2-3 | $3,000-15,000/mo |
| **In-house hire** | Dedicated, aligned execution | Tier 3-4 | $60,000-120,000/year |
| **In-house team** | Full control, maximum output | Tier 4 | $200,000-500,000+/year |
