| 3-5 | <!-- "Wow" moment --> | | |

**Demo Anti-Patterns (Don't Do This):**
- Showing settings or admin panels first (nobody cares)
- Walking through features linearly instead of telling a story
- Demoing features the competitor also has (no differentiation)
