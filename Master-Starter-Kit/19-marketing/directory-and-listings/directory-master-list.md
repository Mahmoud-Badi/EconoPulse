## Quarterly Maintenance Checklist

Every 3 months, review all live listings:

- [ ] Update product description if features have changed
- [ ] Replace screenshots with current UI
- [ ] Update pricing information
- [ ] Respond to any new reviews (positive and negative)
- [ ] Check for broken links in listing
- [ ] Update team/company information
- [ ] Add any new integrations or features to listing
- [ ] Review competitor listings for positioning changes
- [ ] Claim any unclaimed profiles on new directories
- [ ] Remove or update listings for deprecated features
