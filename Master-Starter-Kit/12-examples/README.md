# Section 12: Examples

> **Purpose:** Filled-in reference examples showing the expected output format for every major deliverable in the kit. These are read-only references -- generate fresh content for your project, do not copy examples.
> **Used by:** Any step where you need to see what a completed deliverable looks like.

Examples use a fictional project ("TaskFlow" -- a project management SaaS for small agencies) to demonstrate the expected depth, structure, and tone. When generating your own project files, use these as quality benchmarks, not as content to clone.
