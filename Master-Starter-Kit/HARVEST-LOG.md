# Harvest Log

Chronological record of all improvements applied to the Master Starter Kit from project feedback via `/kit-harvest`.

## Format

Each entry follows this structure:

```
### [DATE] - KF-ID | Category | Severity

**Source:** N project(s)
**Applied to:** `path/to/kit/file.md`
**Summary:** What was added or changed
**Candidate IDs:** KF-PROJ1-001, KF-PROJ2-003
```
