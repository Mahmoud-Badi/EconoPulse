## Recovery Procedures

If something goes wrong during the demo:

| Issue | Recovery |
|-------|----------|
| Demo environment is down | Switch to pre-recorded video at `{{DEMO_RECORDING_PATH}}` |
| Data looks wrong | Reset seed data: `{{DEMO_RESET_COMMAND}}` |
| Feature throws error | Skip to next act, note it, follow up after |
| Slow performance | Acknowledge it, explain it is a demo environment, move on |
| Unexpected question | "Great question — let me show you that after the main walkthrough" |
