## Example Briefs (for reference)

### Minimal brief (skill will infer the rest)
```
A portfolio site for a freelance brand designer based in London.
Dark mode by default, minimal aesthetic, photography-forward.
Pages: Home, Work, About, Contact.
```

### Detailed brief
```
SaaS landing page for "Flowly" — an AI-powered project management tool
targeting indie developers and small dev teams (2-10 people).

Target audience: developers who are frustrated with Jira's complexity
and want something that "just works" without setup overhead.

Brand: modern, slightly playful, very focused. Think Linear meets Notion.
Primary color: indigo. Font: clean sans-serif.

Pages: Home (hero + features + pricing + testimonials + CTA),
Pricing (3 tiers: Free/Pro/Team), Blog, Changelog, About.

Features: newsletter signup on home, contact form, blog with categories.
No auth needed — marketing site only.

Third-party: Crisp for chat, Plausible for analytics, Stripe for pricing display only.
```
