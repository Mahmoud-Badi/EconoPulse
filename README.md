# EconoPulse

Bloomberg Terminal-style world economy dashboard.

## Setup

1. `cd backend && npm install`
2. `cp .env.example .env` (keys already filled)
3. `npm run dev` (starts on port 5000)

4. `cd frontend && npm install`
5. `npm run dev` (starts on port 3000)

## API Keys Used

- **Finnhub**: stocks, forex, sentiment, news
- **Alpha Vantage**: macro economic data, commodities
- **CoinMarketCap**: crypto prices and market data
- **NewsAPI**: economic and business headlines
